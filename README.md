# FUTURE_DS_02 – Customer Retention & Churn Analysis

## Project Overview

This project focuses on customer retention and churn analysis for a subscription-based business. The objective was to identify customer churn patterns, understand retention drivers, and provide actionable recommendations to improve customer retention and reduce customer loss.

## Tools Used

* Power BI
* Microsoft Excel
* GitHub

## Dataset Overview

The dataset contains customer information including:

* Customer ID
* Contract Type
* Internet Service
* Payment Method
* Monthly Charges
* Tenure
* Churn Status

The data was analyzed using Power BI to identify factors influencing customer retention and customer churn.

## Dashboard Features

The dashboard includes:

* Total Customers KPI
* Churned Customers KPI
* Retained Customers KPI
* Customer Churn Distribution
* Churn by Contract Type
* Churn by Internet Service
* Churn by Payment Method
* Customer Tenure Analysis

## Key Findings

* The overall customer retention rate was approximately 73.5%.
* The overall churn rate was approximately 26.5%.
* Customers on month-to-month contracts showed significantly higher churn rates than customers on long-term contracts.
* Customers using fibre optic internet services experienced higher churn levels compared to other service categories.
* Electronic check payment methods were associated with higher customer churn.
* Customers with longer tenure demonstrated stronger retention patterns.

## Recommendations

1. Encourage customers to move from month-to-month contracts to long-term contracts through loyalty incentives.
2. Improve customer engagement strategies for customers identified as high churn risks.
3. Review customer experience and pricing structures for fibre optic service customers.
4. Introduce retention campaigns targeting customers with shorter tenure periods.
5. Offer rewards and loyalty programs to increase long-term customer commitment.
